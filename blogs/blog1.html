HackerRank: "Hello World!" in C

Posted on May 9, 2023

In this blog, we will discuss the solution of hackerrank problem "Hello World! in C and how to write a program for it.

Code Example

Let's take a look at a C program that takes user input and prints it to the console:

#include <stdio.h>
#include <string.h>
#include <math.h>
#include <stdlib.h>

int main() 
{
    char s[100];
    scanf("%[^\n]%*c", &s);
    printf("Hello, World!");
    printf("\n%s",s);
    return 0;
}

Explanation

The main function is where the program execution begins. Inside the main function, we define a character array 's' that can hold up to 100 characters.

Next, we use the 'scanf' function to read a string from the user input until a newline character is encountered. The '%[^\n]' format specifier tells 'scanf' to read all characters until a newline character is encountered, while the '%*c' format specifier tells 'scanf' to read and discard the newline character. The '&' before the 's' variable is used to pass the address of the s array to the 'scanf' function.

Then, we print the string "Hello, World!" to the console using the 'printf' function. Finally, we print the input string entered by the user using the '%s' format specifier, which tells 'printf' to print the string stored in the 's' array.
