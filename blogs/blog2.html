Problem Statement

The problem requires you to take three lines of input from the user:

  1. A single character.
  2. A string of up to 20 characters.

Explanation

The 'main' function begins by declaring three variables:

  • 'ch' is a 'char' variable that can hold a single character.
  • 's' is a 'char' array that can hold a string of up to 20 characters.
  • 'sen' is a 'char' array that can hold a sentence of up to 100 characters.

Next, the 'scanf' function is used to read in the input. '%[^\n]'' is used as the format specifier to read in a line of input until a newline character is encountered. The '%*c' part is used to consume the newline character left in the input buffer by the previous 'scanf' call. The '&' operator is used to pass the address of the variable to 'scanf' so that it can store the input at that address.

After reading in the input, the 'printf' function is used to print out the input. '%c' is used as the format specifier to print out the single character in 'ch', while '%s' is used to print out the strings in 's' and 'sen'. The '\n' character is used to print each input on a separate line.

Finally, the 'return 0' statement is used to end the program.
